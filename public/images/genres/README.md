# 店舗業態の画像ファイル

このディレクトリには、簡易シミュレーション Q1（メインジャンル選択）で使用する店舗業態の画像を配置してください。

## 必要な画像ファイル

以下の12個の画像ファイルが必要です。推奨サイズ: 400x400px 以上

1. **izakaya.jpg** - 居酒屋・酒場の外観または店舗イメージ
2. **japanese.jpg** - 和食・定食（定食プレートや和食料理）
3. **yakitori.jpg** - 焼き鳥・串焼き（焼き鳥を焼いている様子や串盛り）
4. **yakiniku.jpg** - 焼肉・ホルモン（焼肉プレートまたは焼いている様子）
5. **ramen.jpg** - ラーメン・つけ麺（ラーメンまたはつけ麺の写真）
6. **curry_ethnic.jpg** - カレー・エスニック（カレーライスまたはエスニック料理）
7. **italian.jpg** - イタリアン（パスタまたはピザの写真）
8. **french.jpg** - フレンチ・ビストロ（フレンチ料理または店舗内観）
9. **cafe.jpg** - カフェ・喫茶（カフェの内観またはコーヒー）
10. **sweets.jpg** - スイーツ・デザート（ケーキやデザート）
11. **bar.jpg** - バー・ワインバー（バーカウンターまたはカクテル）
12. **bistro.jpg** - 多国籍・創作系（多国籍料理または創作料理）

## 画像の要件

- ファイル形式: JPG または PNG
- 推奨サイズ: 400x400px 以上
- アスペクト比: 正方形（1:1）が望ましい
- ファイルサイズ: 500KB以下を推奨

## 注意事項

画像を配置後、開発サーバーを再起動すると反映されます。
